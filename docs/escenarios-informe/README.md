# Escenarios De Validacion

Esta carpeta contiene escenarios documentados en formato Markdown para sustentar, en un informe, que los procesos principales del proyecto web funcionan correctamente.

## Escenarios incluidos

1. `01-autenticacion-y-acceso.md`
   Valida el inicio de sesion y el acceso segun permisos.

2. `02-importacion-general.md`
   Valida la vista previa e importacion de data general desde Excel.

3. `03-gestion-cartera-y-reportes.md`
   Valida la consulta de cartera, el registro de acciones y la exportacion de reportes.

## Uso sugerido

- Completar la seccion `Resultado obtenido` con lo observado durante la prueba.
- Adjuntar capturas o archivos exportados en la seccion `Evidencia`.
- Marcar la `Conclusion` como `Conforme` o `No conforme`.

## Flujo funcional cubierto

Los escenarios cubren un flujo real del sistema:

`Inicio de sesion -> acceso por permisos -> importacion de datos -> gestion de cartera -> exportacion de reportes`
